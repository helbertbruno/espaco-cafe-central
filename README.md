# Espaço Café Central — Site Institucional

Site institucional desenvolvido para o **Espaço Café Central**, café e espaço de eventos localizado no centro de Juiz de Fora, MG.

Projeto real desenvolvido como freelance, com foco em design premium, responsividade e integração com plataformas de delivery e atendimento via WhatsApp.

---

## 📄 Páginas

- **Home** — apresentação do espaço, ambientes, cardápio e eventos
- **Cardápio** — narrativa visual dos itens do cardápio com botão flutuante para cardápio digital
- **Reservas** — formulário de reserva integrado ao WhatsApp
- **Delivery** — links diretos para o cardápio digital (Goomer) e iFood
- **Evento** — apresentação dos espaços para eventos com formulário de contato via WhatsApp
- **Promoções** — promoções semanais com layout editorial

---

## 🛠️ Tecnologias

- HTML5
- CSS3 (responsivo, mobile-first)
- JavaScript (vanilla)
- Font Awesome
- Google Fonts (Cormorant Garamond + Jost)

---

## ✨ Funcionalidades

- Nav fixa com menu hamburguer para mobile
- Integração com WhatsApp para reservas e eventos
- Links diretos para iFood e Goomer
- Layout responsivo para desktop, tablet e mobile
- Seções com imagens em fullscreen hero
